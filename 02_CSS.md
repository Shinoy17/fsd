# Cascading Style Sheets

## Introduction
* Why CSS?
* External, Internal, Inline
* Syntax
  - Selector { property1: value; property2: value; }
  - Example: p { color: blue; text-align: center; }
  
## CSS Selectors
* The **element** selector
  - p { color: blue; text-align: center; }
* The **id** selector
  - #first { text-align: right; }
*  The **class** selector
  - .group { color: blue; }
